#### 0417

##### static

- static 멤버는 클래스당 하나만 생성
- static 멤버는 클래스 이름으로 접근 가능
- 객체의 멤버로 접근 가능
- 허나 non static 멤버는 클래스로 접근이 안된다.
- 전역변수를 만들 때 사용한다.
- static 메소드는 오직 static 멤버만 접근 가능
- static 메소드는 non static 멤버를 사용 불가

##### 객체 소멸

- 자바는 free() 처럼 객체 소멸 연산자가 없다
- 객체 소멸은 jvm의 고유한 역할
